climbto350 (iOS and iPadOS)

Climbto350 is an aviation job board app. It allows you to browse, search, and apply for all sorts of aviation jobs across all parts of the globe.

I joined the climbto350 team and was tasked with building an iOS app from scratch in June 2023. I started by building a REST API in Python FastAPI to communicate with the existing website's SQL Server database. I then called on the API within the app to sync it with the website. The app itself was created using SwiftUI and some minor UIKit. I released climbto350 to the App Store in July 2024.

So far, the app has:

  • Gained 8k+ downloads
  • Produced over $100k in revenue
  • Gained an overall 5 star rating

Browse Jobs

Users can browse through and search for jobs easily. All recent searches are saved locally.

Apply For Jobs

Users can freely apply to jobs using their climbto350 resume and cover letter.

Create Resume

At any time, users can create or update their existing climbto350 resume. Resumes are downloadable and can be used to natively apply to jobs on the board or anywhere else.

Save Jobs

Jobs can be saved to view and apply later. Users can then easily view all of their saved and applied jobs.
